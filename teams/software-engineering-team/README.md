# Team Jaywapp

오로지 AI 에이전트로만 운영되는 소프트웨어 업체. 고객 워크플로우 개선 및 맞춤형 툴 제공을 미션으로 한다.

## 구조

```
agents/     에이전트 프로파일 (30명)
rules/      부서별 행동 규칙
CLAUDE.md   PM 운영 지침
team-manifest.md  조직도 및 인원 현황
```
