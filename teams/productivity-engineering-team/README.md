# productivity-engineering-team

업무 마찰 제거, 생산성 향상 플랫폼 제공, AI Transformation(AX) 가속을 목표로 하는 AI 에이전트 팀.

## 팀 목표

- 반복 작업 자동화로 개발·협업 생산성 향상
- 내부 플랫폼 및 도구 운영
- AX(AI Transformation) 가속 지원

## 구조

```
productivity-engineering-team/
├── Team/
│   ├── purpose.md         # 팀 목표 및 KPI
│   ├── members/           # 에이전트 담당자 정의
│   └── parts/             # PM / Engineering / Platform / AX / QA / Operations (6개 파트)
├── rules/                 # 일반·개발·문서·운영·보안 규칙
├── wiki/                  # 파트별 내부 지식 문서
├── docs/templates/        # 프로젝트 요청·최종 보고서·운영 이슈 템플릿
├── projects/              # 진행 중 프로젝트
├── skills/                # 재사용 자동화 스킬
├── plugins/               # 플러그인
└── start.md               # 팀 운영 시작 문서 (요청→검증→보고 프로세스)
```

## 시작하기

`start.md`를 읽고 요청 접수 → 검증 → 보고 프로세스를 따릅니다.
