# 목표
## 업무 마찰 제거 (Reduce Friction)
: 구성원들이 시간을 낭비하는 요소 제거

### 예시
- 반복 작업 자동화
- 수작업 프로세스 제거
- 툴 간 연결
- 환경 세팅 간소화
- 정보 접근성 향상

### KPI 예시
- 작업 소요시간 감소
- 반복 업무 자동화율
- 수작업 단계 감소

## 생산성 향상 플랫폼 제공
: 사람이 더 잘 일하게 만드는 기반

### 예시
- 내부 툴 개발
- 빌드/CI/CD
- 파이프라인 개선
- 공통 플랫폼 제공
- 협업 시스템

### KPI 예시
- 빌드 성공률
- 평균 빌드 시간
- 툴 사용률
- 셋업 시간 감소

## AX(AI Transformation) 가속
- AI를 실제 업무 흐름에 녹이는 것

### 예시
- AI 코드 리뷰
- 크래시 분석 자동화
- 문서 생성
- 업무 보조 에이전트
- CL 생성 지원

### KPI 예시
- AI 활용률
- 절감 시간
- 자동 생성 비율

## 개발 및 협업 품질 향상
- 속도만이 아니라 품질도

### 예시
- 휴먼 에러 감소
- 안정적 빌드
- 검증 자동화
- 표준화

### KPI 예시
- 빌드 실패 감소
- 반복 이슈 감소
- 오류 탐지율
