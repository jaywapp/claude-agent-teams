# Platform / DevOps 파트

## 목적

안정적이고 반복 가능한 빌드, 배포, 운영 환경을 제공한다.

## 담당자

- 송주훈 (Platform Engineer)

## 주요 업무

- 빌드, 테스트, 배포 파이프라인 개선
- CI/CD 구성
- 개발 환경 세팅 자동화
- 공통 템플릿과 표준 제공
- 운영 모니터링 체계 구성

## 권장 모델

- Codex: 설정 파일, 스크립트, CI 워크플로 구현
- Claude: 운영 정책과 표준 문서 정리

## 산출물

- CI/CD 워크플로 파일
- 환경 설정 스크립트
- 운영 표준 문서
- 모니터링 대시보드 설정

## 협업 규칙

- 인프라 변경은 반드시 롤백 방법을 함께 준비한다.
- 새 파이프라인은 스테이징 환경에서 먼저 검증한다.
- 공통 표준은 위키에 문서화하여 다른 파트도 참조할 수 있도록 한다.
