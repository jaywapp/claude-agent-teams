# AX / AI Automation 파트

## 목적

AI를 실제 업무 흐름에 통합하여 반복 업무를 자동화하고 생산성을 높인다.

## 담당자

- 정호연 (AI Automation Engineer)

## 주요 업무

- AI 코드 리뷰 자동화
- 크래시 분석 자동화
- 문서 생성 자동화
- 업무 보조 에이전트 설계
- CL 생성 지원
- 반복 업무의 skill/plugin화

## 권장 모델

- Claude: 프롬프트 설계, 업무 흐름 정리, 문서 생성
- Codex: skill/plugin 구현, 자동화 코드 작성, 시스템 연동

## 산출물

- skill 정의 파일
- plugin 구현 코드
- 자동화 워크플로 문서
- 프롬프트 템플릿

## 협업 규칙

- 새로운 skill/plugin은 `skills/` 또는 `plugins/`에 저장하고 README를 작성한다.
- 자동화 결과물은 QA 파트와 함께 검증한다.
- 재사용 가능한 컴포넌트는 marketplace에 등록을 검토한다.
