# 보고 형식 및 주기

## 일간 보고 (PM 작성, 매일 업무 종료 시)
저장 경로: `report/daily/YYYY-MM-DD.md`

템플릿:
```
# 일간 보고 — YYYY-MM-DD

## 오늘 완료된 작업
- [프로젝트명] [부서]: [내용]

## 진행 중인 작업
- [프로젝트명] [부서]: [내용] (예상 완료: N일)

## 이슈 및 블로커
- [이슈 내용] / 해결 방법: [내용]

## 내일 계획
- [계획]

## 토큰 사용 현황
- 오늘 총 사용: 약 [N]K 토큰
```

## 주간 보고 (PM 작성, 매주 월요일)
저장 경로: `report/weekly/YYYY-WXX.md`

템플릿:
```
# 주간 보고 — YYYY 제XX주

## 이번 주 성과
- [프로젝트명]: [주요 성과]

## 진행 중 프로젝트 현황
| 프로젝트 | 단계 | 진행률 | 예상 완료 |
|---------|------|--------|---------|

## 이슈 및 해결
- [이슈]: [해결 내용]

## 다음 주 계획
- [계획]

## 성과 지표
- 완료 프로젝트: N건
- 고객 만족도: [피드백 요약]
- 토큰 사용량: 약 [N]K
```

## 프로젝트 완료 보고 (PM 작성)
저장 경로: `report/projects/[프로젝트명]/final-report.md`

템플릿:
```
# [프로젝트명] 완료 보고

## 프로젝트 개요
- 고객: [이름]
- 기간: YYYY-MM-DD ~ YYYY-MM-DD
- 목표: [한 줄 요약]

## 최종 산출물
- [산출물 목록 및 경로]

## 성과
- 목표 달성 여부: [달성/미달성]
- 고객 피드백: [요약]

## 회고
- 잘된 점: [내용]
- 개선점: [내용]
- 다음 프로젝트 적용 사항: [내용]

## 수치
- 총 소요 기간: N일
- 총 토큰 사용량: 약 [N]K
- 수정 사이클 횟수: N회
```
